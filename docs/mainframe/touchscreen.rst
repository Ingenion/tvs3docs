Touchscreen
===========
The front panel of the TVS 3.0 frames a touchscreen which allows the user to interact with the RPi5.  In turn, the RPi5 interacts with the XEM board as well as the modules.  

The screen is supplied by waveshare.  The model name is 10.4HP-CAPQLED.

Features include:
   - 10.4inch QLED quantum dot display, capacitive touch, high brightness, 1600x720, optical bonding toughened glass panel, HDMI interface.
   - Supports all versions of Raspberry Pi.
   - Excellent display performance with 178Â° wide viewing angle.
   - Up to 6H hardness toughened glass panel. Optical bonding, display well and dustproof.
   - Color gamut of this QLED display is up to: SRGB 94%, NTSC 94%. Chroma of QLED increases 58.3%, making the color more pure and vivid.
   - Refresh rate is 60Hz.
   - Stereo audio speakers.

Size and Resolution
-------------------
The viewable dimensions of the screen are 237.6mm x 106.92mm, encompassing 1600x720 pixels.

Connectivity
------------
The touchscreen connects to the RPi5 over HDMI, providing video and audio.  The touchscreen also connects to the RPi5 over USB providing user-touch input to the RPi5.
